# MyTask Website

Official website for **MyTask**, my first published Android application.

This website serves as the public website for the app and includes the Privacy Policy required for the Google Play Store, along with support and contact information.

---

## About MyTask

MyTask is an offline-first task management application built for Android.

The app helps users organize their daily tasks while keeping their information stored locally on their device.

Current features include:

- Task management
- Image attachments
- Audio notes
- Speech-to-text
- Reminder notifications
- QR-based vCalendar event sharing
- Offline-first experience
